ES6+ Vanilla JavaScript

Overview

This portfolio uses vanilla JavaScript (ES6+) without any frameworks or heavy dependencies. The focus is on clean, performant code that handles user interactions, audio playback, and local storage.

By avoiding frameworks, the site loads faster, has a smaller footprint, and demonstrates fundamental JavaScript skills that translate across any modern web development stack.

Key Features Used

localStorage API

'termsAccepted', 'true'

audio.pause();
