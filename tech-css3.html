CSS3

Cascading Style Sheets

Overview

CSS3 brings powerful styling capabilities including animations, transitions, grid layouts, flexbox, and custom properties (CSS variables). This portfolio leverages modern CSS3 features to create a brutalist design aesthetic with smooth interactions.

Every animation, layout, and visual effect is crafted using pure CSS3 without relying on heavy JavaScript libraries, ensuring optimal performance and maintainability.

Key Features Used

CSS Variables

Custom properties for consistent theming across the site

:root {
--paper: #f2f0e9;
--ink: #1a1a1a;
--red: #e63946;
--blue: #0077b6;
}

Animations

Keyframe animations for signature, arrows, and effects

@keyframes blurToFocus {
0% { filter: blur(20px); }
100% { filter: blur(0); }
}

Grid Layout

CSS Grid for responsive card layouts and sections

.grid {
display: grid;
grid-template-columns:
repeat(auto-fit,
minmax(300px, 1fr));
}

Flexbox

Flexible layouts for header, footer, and components

.flex-container {
display: flex;
justify-content: space-between;
align-items: center;
}

Implementation in This Portfolio

Brutalist Design System

  • • Heavy Borders: 2-4px solid borders for strong visual hierarchy
  • • Paper Texture: Background image overlay for tactile feel
  • • Hover Effects: Transform and color transitions on interactive elements
  • • SVG Animations: Stroke-dasharray for hand-drawn line effects
  • • Responsive Design: Mobile-first approach with breakpoints

Best Practices Applied

Performance

Hardware-accelerated transforms, minimal repaints

Maintainability

CSS variables for easy theme updates

Accessibility

Sufficient color contrast, focus states
