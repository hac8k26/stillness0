SVG

SVG Graphics

Scalable Vector Graphics

Overview

SVG (Scalable Vector Graphics) is an XML-based vector image format that scales perfectly at any size. This portfolio uses inline SVG for hand-drawn annotations, curved arrows, and the favicon.

SVG animations using stroke-dasharray and stroke-dashoffset create the hand-drawn effect that perfectly complements the brutalist design aesthetic.

Key Features Used

Path Elements

Curved lines using Bézier curves for hand-drawn look

<path d="
M 90,25
C 85,33 78,37 70,39
C 62,41 54,39 50,35
" />

Stroke Animation

Drawing effect using dasharray and dashoffset

stroke-dasharray: 200;
stroke-dashoffset: 200;
animation: draw 2s
ease-out forwards;

Text Elements

Handwriting font text within SVG graphics

<text
x="90" y="20"
font-family="Caveat"
font-size="2rem"
>Click Here</text>

ViewBox

Responsive scaling with viewBox attribute

<svg
viewBox="0 0 180 95"
width="180"
height="95"
>
