Sitemap.xml

XML Sitemap Protocol

Overview

A sitemap is an XML file that lists all important pages of a website, helping search engines like Google discover and index content more efficiently. It provides metadata about each page including last modification date, change frequency, and priority.

This portfolio's sitemap includes the homepage, terms, and privacy pages, making it easier for search engines to crawl and understand the site structure.

Current Sitemap

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
<url>
<loc>https://stillness0.netlify.app/</loc>
<lastmod>2026-03-09</lastmod>
<changefreq>weekly</changefreq>
<priority>1.0</priority>
</url>
<url>
<loc>https://stillness0.netlify.app/terms.html</loc>
<lastmod>2026-03-09</lastmod>
<changefreq>monthly</changefreq>
<priority>0.5</priority>
</url>
<url>
<loc>https://stillness0.netlify.app/privacy.html</loc>
<lastmod>2026-03-09</lastmod>
<changefreq>monthly</changefreq>
<priority>0.5</priority>
</url>
</urlset>

Key Elements Explained

<loc> - Location

The full URL of the page. Must start with protocol (http/https) and end with trailing slash if applicable.

<loc>https://stillness0.netlify.app/</loc>

<lastmod> - Last Modified

Date of last modification in YYYY-MM-DD format. Helps search engines prioritize fresh content.

<lastmod>2026-03-09</lastmod>

<changefreq> - Change Frequency

How often the page is likely to change: always, hourly, daily, weekly, monthly, yearly, never.

<changefreq>weekly</changefreq>

<priority> - Priority

Relative priority of this URL (0.0 to 1.0). Homepage typically gets 1.0, other pages lower values.

<priority>1.0</priority>
