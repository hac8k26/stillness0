Tools & Libraries

External Dependencies

Overview

This portfolio uses a minimal set of external libraries and tools to enhance functionality without adding unnecessary bloat. Each tool serves a specific purpose and is loaded via CDN for optimal performance.

The focus is on lightweight, well-maintained libraries that complement the brutalist design and provide essential features like icons, fonts, and storage.

Core Libraries

Lucide Icons

Beautiful, consistent icon library with 1000+ icons. Used for social media links, music player controls, and UI elements throughout the portfolio.

<script src="https://unpkg.com/lucide@latest"></script>
<i data-lucide="github"></i>
<script>lucide.createIcons();</script>
Version: Latest | Documentation

Google Fonts

Three carefully selected fonts: Space Grotesk (headings), IBM Plex Mono (code), and Caveat (handwriting). Loaded via Google Fonts CDN with optimized weights.

<link href="https://fonts.googleapis.com/css2?
family=Space+Grotesk:wght@300;500;700&
family=IBM+Plex+Mono:wght@400;600&
family=Caveat:wght@700&
display=swap" rel="stylesheet">
