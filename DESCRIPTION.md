# Stillness.ft Portfolio - Project Description

## 📝 Short Description (50 words)

A brutalist-style portfolio website showcasing systems architecture and UI engineering expertise. Features interactive SVG animations, hand-drawn annotations, background music, and a GDPR-compliant terms modal. Built with pure HTML/CSS/JS, optimized for SEO, and deployed on Netlify.

---

## 📄 Medium Description (150 words)

Stillness.ft is a modern portfolio website that embodies the principles of brutalist web design while showcasing technical expertise in systems architecture and UI engineering. The site bridges the gap between low-level kernel development and high-level web interfaces, reflecting the dual nature of the developer's skill set.

Key features include interactive SVG animations with hand-drawn curved lines, an animated signature effect, auto-playing background music with custom controls, and a privacy-focused terms acceptance modal. The design utilizes a minimal color palette (paper, ink, red, and blue accents) with heavy borders and monospace typography to create a distinctive technical aesthetic.

Built without frameworks using semantic HTML5, custom CSS3 animations, and vanilla JavaScript, the site is fully responsive, SEO-optimized with comprehensive meta tags, and includes legal pages for terms of service and privacy policy. The entire experience is designed to be fast, accessible, and memorable.

## 📋 Long Description (500+ words)

### Overview

Stillness.ft is a cutting-edge portfolio website that masterfully combines brutalist design principles with modern web technologies to create a unique digital presence. This project serves as both a showcase of technical skills and a testament to the philosophy that great design doesn't require complexity—it requires precision.

### Design Philosophy

The website embraces brutalism, a design movement characterized by raw, honest aesthetics and functional simplicity. Every element serves a purpose, from the heavy 2-4px borders that frame content to the carefully chosen paper-textured background (#f2f0e9) that provides warmth without distraction. The color palette is deliberately minimal: deep ink black (#1a1a1a) for text and borders, vibrant red (#e63946) for accents and alerts, and professional blue (#0077b6) for interactive elements.

### Technical Architecture

Built on a foundation of pure web technologies, the site demonstrates that powerful user experiences don't require heavy frameworks:

**Frontend Stack:**
- Semantic HTML5 for structure and accessibility
- Custom CSS3 with advanced animations and transitions
- Vanilla JavaScript for interactive features
- Tailwind CSS for utility-first styling
- SVG for scalable graphics and animations

**Typography System:**
The site employs a three-tier typography system:
- Space Grotesk (300, 500, 700) for headings and display text
- IBM Plex Mono (400, 600) for technical content and code
- Caveat (700) for handwritten annotations and signatures

### Interactive Features

**1. Hand-Drawn Annotations**
Custom SVG paths create the illusion of hand-drawn curved lines that guide users to important interactive elements. These annotations use stroke-dasharray animations to simulate real-time drawing, adding an organic touch to the technical aesthetic.

**2. Signature Animation**
The "stillness.ft" signature employs a sophisticated blur-to-focus animation that mimics the experience of text coming into clarity, creating a memorable first impression.

**3. Background Music Integration**
An auto-playing music system (The Last of Us theme) with custom play/pause controls enhances the immersive experience while respecting user preferences through localStorage persistence.

**4. Privacy-First Modal System**
A GDPR-compliant terms acceptance modal appears on first visit, storing consent in localStorage. The modal features the same brutalist design language with clear calls-to-action and links to full legal documentation.

### Content Structure

**Hero Section:**
Bold typography introduces the developer as a "Systems Architect" who bridges low-level kernel logic and high-level web interfaces. Location and social links provide immediate contact options.

**Skills Showcase:**
A dual-column layout presents expertise in both "Kernel Space" (Linux Kernel, C, Rust, POSIX) and "User Space" (Next.js, TypeScript, WebAssembly), emphasizing the full-stack nature of the developer's capabilities.

**Technical Logs:**
Three project cards showcase work, with LOG_01 linking to a live project and LOG_02/03 marked as "Coming Soon." Each card features hover effects and maintains the brutalist aesthetic.

**Footer:**
Social media icons (GitHub, LinkedIn, Email) with hand-drawn "Contact Me" annotation, plus legal links and copyright information.

### SEO & Performance

The site is meticulously optimized for search engines and social media:
- 20+ meta tags including Open Graph and Twitter Cards
- XML sitemap for search engine crawlers
- Robots.txt for crawler instructions
- Canonical URLs to prevent duplicate content
- Semantic HTML for better indexing
- Fast load times (< 2s) with minimal dependencies

### Responsive Design

The layout adapts seamlessly across devices:
- Mobile (< 768px): Single column, stacked cards
- Tablet (768px - 1024px): Adaptive grid
- Desktop (> 1024px): Full 12-column grid system

### Accessibility

- ARIA labels on interactive elements
- Keyboard navigation support
- High contrast ratios for readability
- Semantic HTML for screen readers
- Focus states on all interactive elements

### Legal Compliance

Dedicated pages for Terms of Service and Privacy Policy ensure transparency and legal compliance. The terms modal requires explicit acceptance before site interaction, with clear links to full documentation.

### Deployment & Hosting

Hosted on Netlify with automatic deployments from GitHub, the site benefits from:
- Global CDN for fast worldwide access
- Automatic HTTPS
- Continuous deployment on push
- Custom domain support
- Form handling capabilities

### Future Enhancements

The modular structure allows for easy expansion:
- Additional project showcases in LOG_02 and LOG_03
- Blog integration for technical writing
- Case study pages for detailed project breakdowns
- Contact form with Netlify Forms
- Dark mode toggle
- Multi-language support

### Conclusion

Stillness.ft represents a perfect marriage of form and function—a portfolio that doesn't just describe technical skills but demonstrates them through every pixel, animation, and interaction. It's a living example of how thoughtful design, clean code, and attention to detail can create a memorable digital experience that stands out in a sea of template-based portfolios.
