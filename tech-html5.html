HTML5

HyperText Markup Language

Overview

HTML5 is the latest evolution of the standard that defines HTML. It includes new elements, attributes, and behaviors, and a larger set of technologies that allows the building of more diverse and powerful websites and applications.

This portfolio uses semantic HTML5 markup to ensure accessibility, SEO optimization, and clean code structure. Every element is carefully chosen to represent its content meaningfully.

Key Features Used

Semantic Elements

Using <header>, <section>, <footer>, <nav> for meaningful structure

<header>
<h1>Portfolio</h1>
</header>
<section>
<article>...</article>
</section>

Audio API

Native audio playback with controls for background music

<audio id="bgMusic" loop>
<source src="song.mp3"
type="audio/mpeg">
</audio>

SVG Graphics

Inline SVG for scalable, animated graphics and icons

<svg viewBox="0 0 100 100">
<circle cx="50" cy="50"
r="40" fill="red"/>
</svg>

Accessibility

ARIA labels, semantic markup, and keyboard navigation

<button
aria-label="Play Music"
role="button">
Play
</button>
