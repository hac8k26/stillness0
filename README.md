## 🚀 Quick Start

### Prerequisites
```bash
# No build tools required!
# Just a modern web browser
```

### Installation

```bash
# Clone the repository
git clone https://github.com/hac8k26/stillness.ft.git

# Navigate to directory
cd stillness.ft

# Open in browser
open index.html
# or
python -m http.server 8000
```

### Deployment to Netlify

```bash
# 1. Push to GitHub
git add .
git commit -m "Initial commit"
git push origin main

# 2. Connect to Netlify
# - Go to netlify.com
# - Click "Add new site"
# - Import from Git
# - Deploy!
```

## 🔧 Customization Guide

### 🎨 Change Theme Colors

```css
/* Edit in index.html <style> section */
:root {
    --paper: #f2f0e9;  /* Background */
    --ink: #1a1a1a;    /* Text & borders */
    --red: #e63946;    /* Accent 1 */
    --blue: #0077b6;   /* Accent 2 */
}
```

### 📝 Update Content

```html
<!-- Hero Section -->
```
